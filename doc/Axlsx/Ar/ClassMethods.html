- (Object) acts_as_xlsx(options = {})

defines the class method to inject to_xlsx

Examples:

class MyModel < ActiveRecord::Base
   acts_as_xlsx :exclude => [:id, :created_at, :updated_at], :i18n => 'activerecord.attributes'

Parameters:

  • options (Hash) (defaults to: {}) —

    a customizable set of options

Options Hash (options):

  • exclude (Array, Symbol) —

    a list of columns to exclude from the report

  • include (Array, Symbol) —

    a list of associated models to include in the report

  • i18n (String) — default: default nil —

    The path to search for localization. When this is specified your i18n.t will be used to determine the labels for columns.

  • attributes_only (Boolean) — default: default false —

    By default, acts_as_xlsx will use methods on your model over attributes with the same name. If you want to restrict the behavior to only using active record instance attributes when generating xlsx set this to true

  • A (Array, Symbol) —

    list of method names that will be called. the name of the method will be the column. The return value will be applied for each row.
